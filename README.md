# SkillHub

## How to Use

- `general`：通用工作流、Agent 方法论、提示词增强
- `research`：论文阅读、引用管理、科研写作、LaTeX 文档
- `development`：代码开发、调试、Agent 行为管理

ps: 不要全相信任何AIGC，请人工复核。
